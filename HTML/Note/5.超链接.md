#### 超链接

```html
<a href="链接地址">文本或图片</a>                 //默认在当前窗口打开
<a href="链接地址" target="_blank">文本或图片</a> //默认在新的窗口打开
```



#### 外部链接和内部链接

外部链接指向外部网站，引用别人文章之类的

内部链接指向**自身的网站页面**，如文章翻页

内部链接用的是**相对路径**



#### 锚点链接

当页面内容过多过长，通过点击锚点链接，可以直接滑动到当前页面的那一部分

```html

    </div>
    ......<br/>

    ......<br/>
    
    <div id="article">
        <h3>推荐文章</h3>
        <ul>
            <li>文章1</li>

            <li>文章3</li>
```
