#### 文本标记

|    标签    |   说明   |
| :--------: | :------: |
| **strong** | **粗体** |
|   **em**   | **斜体** |
|  **sup**   | **上标** |
|  **sub**   | **下标** |
|     s      |  中划线  |
|     u      |  下划线  |
|    big     | 大号字体 |
|   small    | 小号字体 |

**水平线 <hr/>            添加一条分割线**

**分区 <div>......</div>              将代码分区，使代码更有逻辑，便于CSS操作**

**空格与缩进**

​        **`&nbsp;` : 约为1/3个汉字大小**

​        **`&emsp;` : 约为一个汉字大小**

**列表**

有序列表：<ol>...</ol>

无序列表：<ul>...</ul>

定义列表：<dl> 定义列表、<dt>定义名词、<dd>定义描述

格式：

```html
<ol type='a'>
    <li>第一行</li>
    <li>第二行</li>
    ......
</ol>
ol默认阿拉伯数字排序，可在type调整
<ul>
    <li>第一行</li>
    <li>第二行</li>
    ......
</ul>
ul默认实心圆，可用type调整。circle空心圆、square实心正方形
<dl>
    <dt>名词</dt>
    <dd>对该名词的描述</dd>
</dl>
在高级列表中会用到
```
